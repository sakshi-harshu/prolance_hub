# ProLance Hub – Login System (Milestone 1)

This project is part of my internship milestone submission. It implements a login system using Django.

## Features
- User login page
- Authentication using Django
- Clean and responsive login UI
- Template-based frontend
- Organized Django project structure

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite (default Django database)
